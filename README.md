# GeoInsight — 大宗商品地缘政治分析平台

实时追踪 20 种大宗商品行情，结合地缘政治事件与宏观经济数据，帮助用户理解价格波动背后的驱动因素。

**在线体验**：`http://43.160.241.35:9180`

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TypeScript + Vite + Tailwind CSS |
| 可视化 | Apache ECharts（K 线、热力图、时间轴等） |
| 后端 | FastAPI (Python) + async/await + SSE |
| 数据库 | PostgreSQL 15 |
| 缓存 | Redis 7（行情缓存 + 分布式锁 + 会话存储） |
| AI | DeepSeek（新闻情感分析 + 市场日报 + 对话助手） |
| 部署 | Docker Compose（四容器编排） |

## 功能模块

### 行情看板
- 20 个品种实时价格、涨跌幅、成交量总览
- 单品种 K 线图（OHLCV + MA5/20/60 + 成交量），支持缩放
- 盘中 3 秒级实时推送（SSE）

### 多维分析
- **品种对比**：多品种归一化价格走势对比（time 轴，自适应数据范围）
- **相关性矩阵**：多品种价格相关性热力图
- **宏观对比**：宏观指标（DXY / 国债利率 / BDI 等）与商品价格双轴对比
- **价格比值**：金银比、铜金比等品种间比值走势
- **波动率分析**：历史波动率走势
- **情绪分析**：基于 AI 的市场情绪面板

### 事件时间轴
- 地缘政治事件标注叠加在价格曲线上
- 蓝→紫→红渐变折线 + 虚线连接卡片
- 支持事件回测：选择历史事件，回测前后 N 天内相关品种表现

### 新闻聚合
- 18 路 RSS 源全天候抓取（CNBC / FT / Google News 等）
- AI 自动分析：摘要、情感判断、品种关联、重要度评级

### AI 助手
- 基于 DeepSeek 的流式对话，支持行情问答与地缘分析
- 自动生成每日市场报告

## 数据来源

| 数据 | 来源 |
|------|------|
| 期货日线 | 新浪期货 via akshare |
| 实时行情 | 东方财富 push2 批量接口 |
| 宏观指标 | akshare（DXY / USDCNY / US10Y / CN10Y / BDI） |
| 新闻 | 18 个 RSS 订阅源 |
| 地缘事件 | 人工维护事件数据库 |

## 数据更新

| 任务 | 周期 | 说明 |
|------|------|------|
| 实时行情 | 3 秒 | 交易时段通过东方财富批量获取，SSE 推送前端 |
| 日线补全 | 5 分钟 | 新浪全量日线数据拉取 |
| 新闻抓取 | 15 分钟 | 18 路 RSS 源抓取 + 去重 |
| AI 分析 | 16 分钟 | 未分析新闻自动调用 DeepSeek |
| 宏观指标 | 6 小时 | DXY / 汇率 / 国债利率等 |
